2Projects

Projects are the primary unit of work in The Core. Each project has its own tasks, posts, group chat, file library, and team roster.

Opening a project

  1. Click Projects in the sidebar.
  2. Click any project card to open it.
  3. Use the ← All projects link at the top of the page to return to the list.

Project tabs

TabContents
OverviewProject mandate, description, status, and key metadata
TasksTo-do items with assignees, priorities, and due dates
PostsLong-form announcements and updates from the team
MessagesThe project's group chat channel
FilesUploaded documents organised into folders
TeamMembers assigned to this project
PhasesMilestone phases with deliverables (instructor-managed)

Creating a project

  1. Click Projects in the sidebar.
  2. Click + New Project.
  3. Enter a name, description, colour, and icon.
  4. Click Create.
Note

Only Admins and Owners can create and archive projects. Members can view and contribute to projects they are assigned to.

3Tasks

Tasks track individual pieces of work. Create them inside a project or view all your assigned tasks in aggregate from My Tasks.

Creating a task

  1. Open a project and go to the Tasks tab.
  2. Click + New Task.
  3. Enter a title, then optionally set an assignee, priority, due date, and description.
  4. Click Save.

Task priorities

PriorityWhen to use
CriticalBlocking — must be resolved immediately
HighImportant — complete before lower-priority items
MediumStandard priority (default)
LowNice-to-have — complete when capacity allows

Task statuses

StatusMeaning
To DoNot yet started
In ProgressActively being worked on
In ReviewAwaiting feedback or approval
DoneComplete
Tip

Use My Tasks in the sidebar to see every task assigned to you across all projects, sorted by due date.

4Inbox & Messaging

The Inbox handles direct messages (DMs) and project group chats. Messages support markdown formatting and AI-powered analysis.

Starting a direct message

  1. Click Inbox in the sidebar.
  2. Click the compose icon or search for a person by name.
  3. Type your message and press Enter to send.

Sending a message in project chat

  1. Open a project and go to the Messages tab.
  2. Type your message in the input bar at the bottom.
  3. Press Enter to send, or Shift + Enter for a new line.

Message actions

Hover over any message to reveal the action toolbar:

ReactClick the 😊 icon and select an emoji
ReplyClick Reply to quote the message in your response
EditClick the pencil icon (own messages only)
DeleteClick the bin icon (own messages only)

5Library & Templates

The Library stores shared documents accessible to the whole organisation. Templates provides reusable starting points for common document types.

Creating a document

  1. Click Library (or Templates) in the sidebar.
  2. Open or create a folder by clicking + New Folder.
  3. Inside the folder, click + New Document.
  4. Write your content and click Save.

Using a template

  1. Click Templates in the sidebar.
  2. Open the relevant folder and click a template card.
  3. Click Use Template to create a new document pre-filled with the template content.
Note

Templates can only be created and edited by Admins and Owners. Members can use templates but cannot modify them.

6Team & People

The Team page lists every member in your organisation. Profile cards provide quick access to contact information and a one-click route to messaging.

Viewing a profile card

Click any avatar or @mention anywhere in the app. The profile card displays:

  • Full name, job title, and organisation
  • Bio (if the user has set one)
  • Number of projects the person belongs to
  • A Message button to start a direct message

Updating your profile

  1. Click Settings in the sidebar.
  2. Select Profile.
  3. Update your name, job title, bio, or profile photo.
  4. Click Save.

7Settings & Administration

Administrators have access to organisation-wide settings, including the Basecamp migration pipeline for importing historical data.

Connecting Basecamp

  1. Click Settings → Basecamp.
  2. Click Connect Basecamp and authorise via OAuth.
  3. Select the projects to import from the list.
  4. Click Import selected and wait for the process to complete.

Backfill tools

Each tool is collapsed by default. Click the › chevron to expand it, then click Run. All tools are safe to run multiple times.

ToolPurpose
Import Message Board PostsPulls Basecamp message board posts into Posts
Import To-dos as TasksConverts Basecamp to-do lists to project tasks
Import Project Campfire ChatsPulls project group chat history into Messages
Add Members to ProjectsSyncs project membership from Basecamp activity
Import Private MessagesBrings in Basecamp 1-on-1 and group DMs
Populate Project MandatesFills the mandate tab from project descriptions
Fix: Projects hiddenResets imported-archived projects to Active
Clear Imported MessagesRemoves all imported chat data (irreversible)
Note

Clear Imported Messages cannot be undone. Use it only to clean up a failed import before re-running.
